एक बाज़ (kestrel) पूरी सटीकता से देखता है, फिर झपट्टा मारता है। Kaestral आपकी फ़ुटेज देखता है, हर शब्द
सुनता है, और काट देता है। बस लिखिए — *"बोरिंग हिस्से हटाओ, कैप्शन जोड़ो, हुक पर ज़ूम-इन करो"* — और अपनी
टाइमलाइन को बदलते हुए देखिए।

आपके पास पहले से मौजूद **Claude Code** (मुफ़्त) के साथ काम करता है, या इन-ऐप चैट के साथ। एक असली प्रो
टाइमलाइन पर बना है — इसलिए जब AI का काम पूरा हो जाए, तब भी आप हर फ़्रेम को छू सकते हैं। **सब कुछ लोकल
चलता है — आपका वीडियो आपकी मशीन से बाहर कभी नहीं जाता।**

## शुरुआत करें

सबसे तेज़ तरीका — अपने पास पहले से मौजूद Claude Code को Kaestral से जोड़ें। **एक कमांड, ज़ीरो कॉन्फ़िग:**

```bash
claude mcp add kaestral -- npx kaestral
claude
```

Claude Code खुद `npx kaestral` चलाता है और उससे stdio पर बात करता है — कोई अलग प्रोसेस नहीं, कोई पोर्ट
याद रखने की ज़रूरत नहीं। फिर बस कहिए:

> *"get_timeline, फिर demo.mp4 के खामोश हिस्से काटो, कैप्शन जोड़ो, और एक्सपोर्ट कर दो।"*

पूरा डेस्कटॉप ऐप चाहिए? **[Windows के लिए डाउनलोड करें →](https://github.com/prabindersinghh/Kaestral-pro/releases/latest)** — इंस्टॉलर पर डबल-क्लिक करें; एडिटर और उसका इंजन अंदर ही चालू हो जाते हैं (कोई टर्मिनल नहीं, कोई npm नहीं)।

## यह क्या करता है

### 🎬 AI-संचालित। आप निर्देशन करें, यह एडिट करे।
सामान्य भाषा में एडिट बताइए और उसे अपनी टाइमलाइन पर होते देखिए — काटना, कैप्शन, ज़ूम-इन, ग्रेड, टाइटल।
हर क्रिया एडिटर के टूल्स से होकर गुज़रती है, इसलिए नतीजा एक असली, एडिट-योग्य प्रोजेक्ट होता है — कोई
ब्लैक-बॉक्स एक्सपोर्ट नहीं।

### ⚡ कच्ची फ़ुटेज → पब्लिश-रेडी रील।
20-मिनट की रिकॉर्डिंग डालिए; एक कसा हुआ, कैप्शन वाला, बीट-कट शॉर्ट पाइए। Kaestral **फ़िलर शब्द हटाता है**,
**हुक ढूँढता है**, और **शब्द पर कैप्शन** लगाता है — अपने-आप। भाषण ऑन-डिवाइस ट्रांसक्राइब होता है।

### 📊 बिना एडिटर के SaaS और प्रोडक्ट वीडियो बनाइए।
एनिमेटेड इंट्रो, लोगो रिवील, डेटा-विज़, ट्रांज़िशन स्टिंगर — **एक वाक्य से आर्ट-डायरेक्ट** होकर आपकी
टाइमलाइन पर रेंडर। टेम्पलेट नहीं: एक जुड़ा हुआ LLM `compose_motion` सरफ़ेस से मौलिक मोशन रचता है।

**50 MCP टूल्स।** एक पूरा एडिटर जिसे AI हाथों की तरह चलाता है।

## एक AI कनेक्ट करें

### विकल्प A — Claude Code (मुफ़्त, एक कमांड)
```bash
claude mcp add kaestral -- npx kaestral
claude
```

### विकल्प B — इन-ऐप चैट (आपकी अपनी Anthropic API की)
डेस्कटॉप ऐप खोलें → **Connect AI** → अपनी की पेस्ट करें। Kaestral के अंदर ही चैट करें; एडिट्स विंडो में
लाइव होते हैं।

> **ज़रूरी:** `npx kaestral` के लिए Node/npm और **FFmpeg + ffprobe** PATH पर चाहिए। whisper मॉडल
> (~142 MB) पहली बार में डाउनलोड होता है। Windows इंस्टॉलर सब कुछ बंडल करता है, इसलिए ऐप को इनमें से
> किसी की ज़रूरत नहीं।

## प्राइवेसी — लोकल-फ़र्स्ट

आपकी वीडियो और प्रोजेक्ट फ़ाइलें आपके डिवाइस पर ही रहती हैं। Kaestral उन्हें कभी अपलोड नहीं करता।
ट्रांसक्रिप्शन, फ़्रेम विज़न, बीट डिटेक्शन और रेंडरिंग — सब **लोकल** चलता है। केवल दो चीज़ें आपकी मशीन से
बाहर जाती हैं: आपके **AI प्रॉम्प्ट** (Anthropic को, सिर्फ़ तब जब आप अपनी की के साथ इन-ऐप चैट इस्तेमाल
करें), और आपका **ईमेल** (सिर्फ़ तब जब आप Pro/Max वेटलिस्ट जॉइन करें)। किसी अकाउंट की ज़रूरत नहीं।

---

## Pro और Max (वेटलिस्ट)

- **Pro — जल्द आ रहा है:** AI वीडियो और इमेज **जनरेशन** (प्रॉम्प्ट → आपकी टाइमलाइन पर असली क्लिप),
  **अनुवाद और डबिंग** (हिन्दी + क्षेत्रीय भारतीय भाषाएँ, और आगे), टेम्पोरली-कंसिस्टेंट **अपस्केलिंग**,
  और उन्नत जनरेटिव मोशन।
- **Max —** कुछ बड़ा। जानकारी अभी गुप्त।

ऐप के **✨ Pro** बटन से, या [प्राइसिंग पेज](https://kaestral.com/pricing.html) से जॉइन करें। Basic
हमेशा के लिए मुफ़्त और ओपन-सोर्स है।
